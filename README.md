# NeuroFocus

**NeuroFocus** — приложение для мониторинга усталости и осанки с управлением курсором мыши через жесты. Работает в реальном времени с помощью компьютерного зрения (MediaPipe) и LSTM-модели усталости.

## Возможности

### 1. Мониторинг усталости
- Отслеживание EAR (Eye Aspect Ratio) — степень открытости глаз
- Отслеживание MAR (Mouth Aspect Ratio) — обнаружение зевания
- LSTM-нейросеть для классификации состояния: Awake / Drowsy / Sleeping
- Детекция микросна (длительное закрытие глаз)
- Звуковые и визуальные оповещения при усталости

### 2. Анализ осанки
- Геометрический анализ положения плеч и головы по MediaPipe Pose (33 ландмарка тела)
- Три метрики: высота головы над плечами, асимметрия плеч, угол наклона корпуса
- Классификация: хорошая / средняя / плохая осанка с порогами по нормализованным координатам
- Оповещения при отклонении от нормы (не чаще раза в 30 секунд)

### 3. Управление жестами
- Управление курсором мыши движением ладони
- Адаптивное EMA-сглаживание с защитой от дёрганий
- Калибруемая зона управления (два угла — верхний левый и нижний правый)

| Жест | Действие |
|------|----------|
| Только указательный палец | Движение курсора |
| Кулак (все пальцы сжаты) | Левый клик |
| Указательный + мизинец | Правый клик |
| 3 пальца: указ. + сред. + безым. | Прокрутка вверх |
| 4 пальца: указ. + сред. + безым. + мизинец | Прокрутка вниз |

### 4. Pomodoro таймер
- Настраиваемое время работы и перерывов
- Длинный перерыв после 4 циклов
- Звуковые уведомления при смене фазы

### 5. Калибровка
Последовательная калибровка через единое диалоговое окно (4 шага, каждый можно пропустить):
1. **Лицо** — базовые значения EAR/MAR для текущего пользователя
2. **Осанка** — эталонное положение плеч и головы
3. **Рука** — базовый размер ладони
4. **Зона управления** — обведение рабочей области камеры (левый верхний → правый нижний угол)

### 6. Интерфейс
- Тёмная тема
- Мини-график усталости в реальном времени
- Журнал событий
- Системный трей (сворачивание без выхода)
- Статистика и экспорт данных

## Архитектура

### Обработка видео
`VideoThread` захватывает кадры с камеры и запускает три параллельных потока:
- **face** — EAR/MAR, наклон головы (MediaPipe Face Mesh)
- **hand** — детекция жестов, управление мышью (MediaPipe Hands)
- **heavy_ml** — LSTM усталость + анализ осанки (MediaPipe Pose)

### Анализ осанки
Вместо обученной нейросети используется геометрический анализ по ландмаркам MediaPipe Pose:
- `head_height` = Y-центр плеч − Y-носа (норма 0.20–0.40)
- `shoulder_diff` = |Y левого плеча − Y правого плеча| (норма < 0.05)
- `shoulder_angle` = угол наклона линии плеч в градусах (норма < 8°)

### Управление мышью
Адаптивный контроллер с EMA-сглаживанием и счётчиком выбросов:
- Единичные выбросы (1–2 кадра) — курсор замораживается
- Стабильное смещение (3+ кадра подряд) — телепортация в цель
- После калибровки зоны — нормализация позиции ладони в калиброванных координатах
