# NeuroFocus

**NeuroFocus** — приложение для мониторинга усталости, осанки и управления курсором мыши с помощью жестов. Использует компьютерное зрение (MediaPipe) и PyQt6.

## Возможности

### 1. Мониторинг усталости
- Отслеживание EAR (Eye Aspect Ratio) — открытость глаз
- Отслеживание MAR (Mouth Aspect Ratio) — зевание
- Анализ тренда усталости в реальном времени
- Звуковые и визуальные оповещения при обнаружении усталости

### 2. Анализ осанки
- Детекция наклона головы
- Детекция наклона плеч
- Оповещения при плохой осанке

### 3. Управление жестами
- Управление курсором мыши движением руки
- Левый клик — пинч (большой + указательный)
- Правый клик — указательный + средний
- Перетаскивание — кулак

### 4. Pomodoro таймер
- Настраиваемое время работы и перерывов
- Длинный перерыв после 4 циклов
- Интеграция с мониторингом усталости (автопредложение перерыва)
- Звуковые уведомления

### 5. Темы и интерфейс
- Тёмная и светлая темы
- Мини-график усталости в реальном времени
- Системный трей с фоновой работой

## Установка

### Требования
- Python 3.10+
- Веб-камера

### Установка зависимостей

```bash
pip install -r requirements.txt
```

### Структура проекта

```
neurofocus/
├── analyzers/          # Анализаторы (усталость, осанка, эмоции)
├── controls/           # Управление (жесты, калибровка)
├── core/               # Ядро (конфиг, настройки)
├── detectors/          # Детекторы (лицо, руки, поза)
├── ui/                 # UI компоненты (темы, графики)
├── utils/              # Утилиты (геометрия, звук, БД)
└── windows/           # Окна (главное, настройки, статистика)
```

## Использование

### Запуск
```bash
python -m app.main
```

### Горячие клавиши
| Клавиша | Действие |
|--------|----------|
| G | Включить/выключить управление мышью |
| Esc | Выключить управление мышью |

### Жесты мыши
| Жест | Действие |
|-------|----------|
| Указательный палец | Движение курсора |
| Пинч (большой + указательный) | Левый клик |
| Указательный + средний | Правый клик |
| Кулак | Перетаскивание |

## Настройки

Настройки сохраняются в `neurofocus/config/`:
- `settings.json` — пользовательские настройки
- `calibration.json` — данные калибровки

## Архитектура

### MediaPipe Integration
Используется MediaPipe Tasks Vision для:
- Face Landmarker — 478 landmarks лица
- Hand Landmarker — 21 landmark руки
- Pose Landmarker — 33 landmarks тела

### Анализаторы
- **FatigueAnalyzer** — комбинированный анализ усталости
- **PostureAnalyzer** — анализ осанки по лицу и плечам
- **EmotionDetector** — распознавание эмоций (CNN модель)

### Калибровка
Автоматическая калибровка при первом запуске для установки базовых значений EAR/MAR.

## Разработка

### Структура кода
- Модульная архитектура с разделением на detectors, analyzers, controls
- Единый интерфейс для всех компонентов
- ThemeManager для темной/светлой темы
- SettingsManager для персистентности настроек
